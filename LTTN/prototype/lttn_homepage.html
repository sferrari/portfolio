How to use:

1 Edit the button shapes to have the options you want in the droplist.

2 Edit the values of the text fields to match the button shape options.

tip: tab through the options and use F2 to edit text

 

So what's going on here?

To simulate the droplist functionality, when a user clicks a button shape:

1 The value of the text field is assigned to a variable ("DroplistOption")

2 The text on the droplist label widget is set to the "DroplistOption" variable

 

The text fields are used because Axure 5.6 cannot change the text on a widget to be equal to the text on a button shape, but it can be equal to a variable value.
